# Adaptive Multi-Wavelet Frequency-State Space Network (AMF-SSNet)

## Dataset Sources
The datasets (Synapse, ACDC) is sourced from this Repo, who built [Swin-Unet](https://github.com/HuCaoFighting/Swin-Unet). The reason, for taking dataset from Repo, as the baseline paper [EW-ViT](https://doi.org/10.1109/WACV61041.2025.00889), uses the datasets provided by TransUnet's authors which is already preprocessed.

### Synapse/BTCV Dataset
The Google Drive [Link](https://drive.google.com/drive/folders/1ACJEoTp-uqfFJ73qS3eUObQh52nGuzCd) is provided for this Synapse dataset.

### ACDC Dataset
The Google Drive [Link](https://drive.google.com/drive/folders/1KQcrci7aKsYZi1hQoZ3T3QUtcy7b--n4) is provided for this ACDC dataset.

### ISIC Dataset
The Google Drive [Link](https://challenge.isic-archive.com/data/#2018) is provided for this ISIC 2018 dataset.
